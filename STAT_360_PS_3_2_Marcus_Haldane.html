For this path diagram, one path must be fixed for the Attitudes factor, the Intention factor, and the Norm factor, for a total of 3 fixed paths. Those 3 fixed paths could be any combination of the paths which symbolize a loading from the dimensions to their respective factors. For example, the paths between Attitudes and Attitude1, Intention and Intention1, Norm and Norm1 could all be fixed so that the scale of each of those factors could be properly established.

Exercise 3.07 - Plant Diversity and Environmental Factors (continued)

Part (a)

# Written answer below

There are 12 dimensions and 4 factors contained in this path diagram.

Part (b)

# Written answer below

One path between a dimension and its respective factor must be fixed, for each of the factors in the path diagram. This means that a total of four paths must be fixed to properly establish the scale of each factor. There are a large number of possible combinations of which four paths could be fixed to properly establish the scale of each factor, but a simple example is to fix the paths between soil carbon and Abiotic Stress, animal activity and Disturbance, standing biomass and Community Biomass, & Plant Diversity and species count.
Additionally, the number of data points for this model would be [12 * (12 + 1)]/2 = 78, while the number of parameters to be estimated would be 46. [(12 displayed regression coefficients between dimensions and factors, 6 displayed regression coefficients between factors, 12 hidden regression coefficients between errors and dimensions, 4 hidden regression coefficients between disturbances and factors, and 16 variances on those hidden errors and disturbances) - the 4 fixed paths used to establish scale between factors and their respective dimensions] Since the number of parameters is less than the number of data points, no additional paths need to be fixed.

Part (c)

# Written answer below

Each of the dimensions and factors on the path diagram are dependent variables, so each of those 16 has an error or disturbance. Therefore, there are 16 additional paths that represent regression coefficients hidden in this path diagram.

Part (d)

# Written answer below

Each of the dimensions and factors on the path diagram are dependent variables, so each of those 16 has an error or disturbance. Those 16 errors and disturbances each have a variance, as they are the only independent variables for this path diagram. Thus, there are 16 variances hidden in this path diagram.

Part (e)

# Written answer below

Since this path diagram has 12 dimensions, there are 12 covariances, one for each of the dimensions. Additionally, each of the dimensions has a variance with every other dimension, resulting in an additional 66 variances for a total of 78 unique data points.

Part (f)

# Written answer below

This model is over-identified, as the number of data points is greater than the number of parameters to be estimated. There are 76 data points, and 50 total parameters if none are fixed. However, at least 4 parameters must be fixed, 1 for each of the factors, bringing the number of parameters to be estimated down to 46. The fixing of those parameters does not change the fact that the model is over-identified.
